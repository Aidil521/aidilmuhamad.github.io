My Project

Design and Development Flight Controller for Quadcopter Drone Control

This project is my final project, which focuses on research and development of flying robots, especially in Flight Controller research. The flight controller developed in this research is a position stability control system with the PID method and a drone remote control system using LoRa Ra-02. The system is run with the RTOS multitask method.

Design of a Two-Way Remote Communication System Based on LoRa Ra-02

This project is my independent project, this project focuses on creating a two-way communication protocol system using LoRa technology. The two-way communication system carried out in this project is full-duplex two-way communication. Testing is done by remotely controlling the LED using a push button on 2 devices that are used simultaneously.

Designing an RC Car with Bluetooth Control System Using Arduino Uno R3

This project is my independent project, this project focuses on controlling RC cars using bluetooth remote control via Smartphone. Controlling RC cars using the Arduino Bluetooth Controller application and the components used in this research are the Arduino Uno microcontroller, Shield L293N, 4x DC Motor, HC-SR04, G90 Servo, and Bluetooth HC-05.

Design of a Dual-Axis Solar Tracker Prototype Based on Arduino Uno R3

This project is a group project for my course. This project focuses on a solar panel tracking control system with a two-axis system using servos. The components used in this project are Arduino Uno, Shield L298D, 2x Servo G90 and 4x LDR Sensor. In this group project I am responsible for programming and selecting the components used.
