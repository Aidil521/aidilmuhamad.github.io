My Project

Design and Development Flight Controller for Quadcopter Drone Control

This project is my final project, which focuses on the research and development of flying robots specifically on Flight Controller research. The flight controller developed in this research is a position stability control system and a drone remote control system using LoRa Ra-02.

Design of a Two-Way Remote Communication System Based on LoRa Ra-02

This project is my independent project, this project focuses on creating a two-way communication protocol system using LoRa technology. This research uses 2x ESP32 microcontrollers, 2x LoRa Ra-02 modules and 2x self-designed shields.

Designing an RC Car with Bluetooth Control System Using Arduino Uno R3

This project is my independent project, this project focuses on controlling RC Cars using bluetooth remote control via Smartphone. The components used in this research are Arduino Uno microcontroller, L293N Shield, 4x DC Motor, HC-SR04, G90 Servo, and HC-05 Bluetooth.

Design of a Dual-Axis Solar Tracker Prototype Based on Arduino Uno R3

This project was a group project for my course. This project focuses on a solar panel tracker control system with a two-axis system using a servo. The components used in this project are Arduino Uno, Shield L298D, 2x Servo G90 and 4x LDR Sensor.
